# 創建伺服器
## 在您登入以後
<Steps>
- ### 請至 [資源頁面](https://dash.shdcloud.xyz/dashboard)
- ### 點擊 [創建伺服器](https://dash.shdcloud.xyz/dashboard/create)
- ### 設定伺服器
	- 伺服器名稱
	- [節點]((https://dash.shdcloud.xyz/api/admin/location/get/all)
		- 台灣
		- 美國
	- [伺服器類型](https://dash.shdcloud.xyz/api/admin/egg/get/all)
		- 請看下方
	- CPU (單位為%)
	- 記憶體 (單位為 MiB)
	- 儲存空間 (單位為 MiB)
	<Callout type="warning">請確實填寫</Callout>
- ### 點創建按鈕
- ### 完成了，好欸
</Steps>
